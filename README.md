# skills

[![Skills](https://www.skills.sh/badge/amk-dev/skills)](https://www.skills.sh/skills/amk-dev/skills)

A collection of [Claude Code](https://code.claude.com) skills by [@buriedstupidity](https://x.com/buriedstupidity).

## Install

Install everything:

```sh
npx skills add amk-dev/skills
```

Or install a single skill:

```sh
npx skills add amk-dev/skills/find-session
```

## Skills

| Skill | Description |
| --- | --- |
| [`find-session`](./find-session) | Search your Claude Code conversation history and return `claude --resume` commands for matching past sessions. |
